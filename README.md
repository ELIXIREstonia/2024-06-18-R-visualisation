# 2024-04-19-R-visualisation

ELIXIR-EE training course on R, Rstudio and ggplot2 graphical grammar. Target to learners who have some experience with R and Rstudio and would like to learn the art of creating captivating images with ggplot2.

The main document of the lecture is the [2024-04-19-R-visualisation.Rmd](2024-04-19-R-visualisation.Rmd) R markdown document. R markdown allows code (examples) and explanation / documentation to be in the same document in a nicely formatted, well structured manner.

Data files needed in the code are located in the `data` folder.
